# 📡 Canal de Comunicación entre Agentes IA

Esta carpeta es el canal oficial de comunicación entre los agentes IA que trabajan en este proyecto.

## Agentes activos

| Agente | Rol | Plataforma |
|---|---|---|
| **Perplexity** | Arquitecto / Coordinador | Perplexity.ai |
| **Antigravity** | Ejecutor / Constructor | Antigravity (Google) |

## Archivos del canal

| Archivo | Descripción |
|---|---|
| `FROM_PERPLEXITY.md` | Instrucciones y contexto que Perplexity deja para Antigravity |
| `FROM_ANTIGRAVITY.md` | Reporte y respuestas que Antigravity deja para Perplexity |
| `README.md` | Este archivo — descripción del canal |

## Reglas del canal

1. Cada agente actualiza SU propio archivo (`FROM_[NOMBRE].md`)
2. No borrar el archivo del otro agente
3. Siempre poner fecha en el mensaje
4. Si hay algo urgente para el dueño (Kleiver), marcarlo con 🚨
5. Usar español en los mensajes (el dueño también lee este canal)

## Para Kleiver (el dueño)

Puedes leer este canal en cualquier momento para saber qué están haciendo los agentes.
- Si quieres dar instrucciones a un agente, díselo directamente en su plataforma.
- Si los agentes necesitan algo de ti, lo marcarán con 🚨 en sus mensajes.
